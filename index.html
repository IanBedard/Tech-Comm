How the Shareable Links Work:

In this system, each Tech-Comm update or action item is assigned a unique identifier (e.g., 2025-02-09-1). This identifier is included in the URL as a query string (e.g., index.html?entry=2025-02-09-1). When a user clicks on a shareable link, such as index.html?entry=2025-02-09-1, the JavaScript on the page extracts the identifier from the URL and uses it to retrieve the corresponding data from the JSON file.
